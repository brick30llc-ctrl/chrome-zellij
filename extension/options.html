chrome_zellij settings

Tiles your real Chrome/Brave windows. No iframes — every pane is a full browser you can surf from.

Tip: "normal" windows let you type URLs in each pane. "clean" mode looks tidier but you navigate via the bar's surf box, links, or the command palette.

Keyboard shortcuts

These global shortcuts are owned by the browser — the extension can show them but can't change them. Use the shortcuts page below to set or fix any that say Not set.

Browsers block extensions from opening this page directly, so if "Open" does nothing, paste it into your address bar.

Bar keys (only while the control bar is focused): Ctrl+Space leader, then h j k l move · n new · x close · g layout · / palette · 1–9 jump.
