chrome_zellij — controls

MouseClick a pane or a bottom chip to focus it. Toolbar buttons for new/close/layout.
Ctrl+SpaceEnter LEADER mode (Zellij-style), then press one of:
h j k lMove focus left / down / up / right (also plain arrow keys)
n / xNew pane / close focused pane
gToggle layout: grid ⇄ binary-split (BSP)
1…9Jump focus to pane N
/Open the fuzzy command palette
EscLeave leader / close palette / help
